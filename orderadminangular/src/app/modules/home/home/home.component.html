Order administration

In the upper side you will find a Select control:

1 - Select Catalog option to watch all products and select amount to add to the cart

2 - Select Shoping cart to validate your order

3 - Select Order list to view your orders generated


This project was developed with:

- Angular CLI: 16.2.9

- Node: 16.20.2

- Angular material

- Bootstrap

- fortawesome



To run correctly this project please follow the next instructions:

set the correct port number on the enviroment/enviroment.ts file

In backend, setup the correct name for Server in the connectionstring on appsettings.Development.json
